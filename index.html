Pretty Pictures

Pretty Pictures

November 2018

Motivation

Final project for Intro to Program Design course (COMP 215)
Implemented a genetic algorithm and deployed a web microservice for
generating pretty, as defined by the user, computer graphics and images using Java

Process

Each computer graphic was generated through a series of arithmetic operations on rgb color
values and external image files, which were then formatted into JSON for parsing and reading.
Through ApacheSpark and JavaScript, the user is given an interface that allows them to choose
2 or more default images for crossbreeding and mutation, thus producing prettier children images
with each generation.

Product

The interface and some pretty pictures I was able to make with it.
